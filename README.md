# Juego del Ahorcado de Facturación Universitaria

# Integrante: Miguel Ángel Suárez Prada
Objetivo del Sistema

Desarrollar un juego interactivo del Ahorcado utilizando el lenguaje de programación Python con el propósito de reforzar el aprendizaje de conceptos relacionados con la facturación universitaria de la Universidad Internacional del Ecuador (UIDE). El sistema aplica los principios de la lógica de programación mediante el uso de estructuras condicionales, estructuras repetitivas, funciones y estructuras de datos, proporcionando una herramienta educativa dinámica y de fácil utilización.

# Descripción de las Funcionalidades
El sistema desarrollado permite al usuario interactuar con un juego del Ahorcado enfocado en conceptos de facturación universitaria.
Entre las principales funcionalidades se encuentran:
- Selección aleatoria de palabras relacionadas con la facturación universitaria.
- Presentación de pistas para facilitar el aprendizaje.
- Visualización de la palabra mediante guiones bajos.
- Ingreso y validación de letras por parte del usuario.
- Control de un máximo de seis intentos por partida.
- Verificación de letras correctas e incorrectas.
- Detección de letras repetidas.
- Mensajes de victoria y derrota.
- Posibilidad de reiniciar el juego.
- Organización del código mediante funciones.
- Uso de estructuras de datos como listas, tuplas y diccionarios.
- Implementación de estructuras condicionales (`if`, `elif`, `else`).
- Implementación de estructuras repetitivas (`for`, `while`, `range`).
- Utilización de las instrucciones `break` y `continue`.
- Código documentado mediante comentarios para facilitar su comprensión y mantenimiento.
